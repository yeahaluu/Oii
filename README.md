## 주요 기능

 - 중간 거리 찾기
 - 지하철역 조회
 - 근처 맛집 조회
 - 맛집 분석
<br>


## 주요 기술 스택

 - **Frontend**: Vue.js, Vuetify, HTML/CSS/JavaScript
 - **BackEnd**: Spring Boot, JPA(Hibernate)
 - **Storage**: MariaDB
 - **Deploy**: AWS EC2

<br>
